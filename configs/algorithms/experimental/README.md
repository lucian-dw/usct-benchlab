# Experimental Algorithm Configs

This directory contains optional and retired development configs. They are kept
for reproducibility, but they are not v0.1 mainline defaults.

Mainline release configs remain in `configs/algorithms/`:

- straight-ray CGLS/SIRT/SART
- attenuation SIRT
- bent-ray travel-time surrogate
- rWave travel-time surrogate
- k-Wave FWI full pipeline
- tiny FWI smoke

Configs in this directory cover data-sanity checks, external MATLAB adapter
experiments, and deprecated debug paths.
